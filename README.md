# Implementación de K-Means Clustering con Flask

## Descripción del proyecto

Este proyecto consiste en el desarrollo de una aplicación web en Flask que permite aplicar el algoritmo K-Means Clustering sobre un dataset de clientes interesados en solicitar un crédito de vivienda.

La aplicación permite visualizar el dataset, ejecutar el modelo de agrupamiento, observar la gráfica del método del codo, visualizar los clústeres generados, identificar los centroides y analizar los resultados obtenidos.

## Objetivo

Desarrollar una aplicación web que permita comprender, implementar y visualizar el funcionamiento del algoritmo K-Means Clustering, integrando análisis de datos, preparación de variables, agrupación automática e interpretación de resultados.
