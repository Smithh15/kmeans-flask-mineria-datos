Conceptos principales de K-Means Clustering

K-Means es un algoritmo de aprendizaje no supervisado utilizado para agrupar datos en clústeres. Su objetivo es organizar los registros de tal forma que los elementos dentro de un mismo grupo sean similares entre sí y diferentes a los elementos de otros grupos.

Funcionamiento general del algoritmo

  1. Se define el número de clústeres K.
  2. El algoritmo selecciona centroides iniciales.
  3. Cada registro se asigna al centroide más cercano.
  4. Se recalculan los centroides con base en los datos asignados.
  5. El proceso se repite hasta que los centroides dejan de cambiar significativamente.

Variables utilizadas para el agrupamiento

Para este ejercicio se seleccionaron variables financieras relevantes para analizar perfiles de clientes interesados en créditos de vivienda:

  • ingreso_mensual: representa la capacidad económica del cliente.
  • monto_credito: indica el valor del crédito solicitado.
  • puntaje_crediticio: representa el comportamiento financiero del cliente.
  • cuota_mensual: estima el valor mensual que pagaría el cliente.

Estas variables fueron normalizadas antes de aplicar K-Means, debido a que manejan escalas diferentes. Por ejemplo, el ingreso mensual y el monto del crédito tienen valores mucho más grandes que el puntaje crediticio.

Ejecución del modelo K-Means

Selecciona el número de clústeres K que deseas generar. Para este ejercicio se recomienda utilizar K = 3, ya que el dataset fue construido con tres perfiles generales de clientes: bajo, medio y alto.
